# Marketing Campaign Analysis (SQL)

## Objective
Analyze marketing campaign data to understand what drives customer conversion.

## Dataset
Customer personality / marketing campaign dataset (2240 customers).  
Target variable: `Response` (1 = converted).

## Tools
- SQL Server (SSMS)
- SQL

## Key Results
- Overall conversion rate: **14.9%**
- High spenders conversion rate: **41.9%** (strongest predictor)
- High purchase frequency conversion rate: **23.4%**
- High-income customers conversion rate: **21.5%**

## Files
- `queries.sql` — SQL used for KPI, segmentation, and behavioral analysis
- `insights.md` — business insights and recommendations
